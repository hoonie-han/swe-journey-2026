## What I Learned Today

Box Model

Every HTML element is a box made of 4 layers (inside to outside):

- Content — the actual text/image
- Padding — space inside, between content and border
- Border — the edge line
- Margin — space outside, between this box and other boxes

Key difference

- padding = inner space (content ↔ border)
- margin = outer space (box ↔ other boxes)

Example
div {
border: 3px solid red;
padding: 10px;   /* inner space */
margin: 20px;    /* outer space */
}

Tip

- inline elements (like span) ignore width/height & top/bottom margin
- → use block or inline-block for full box control
