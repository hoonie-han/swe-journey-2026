## What I Learned Today

### CSS Review — Boxing

- An HTML page is ultimately made up of boxes.
- Depending on whether you want the box's width/height to include padding and border, you can set it as a **border-box** or a **content-box**.
  - `content-box`: width/height = content only
  - `border-box`: width/height = content + padding + border
  - In practice, most developers use `border-box`.
- When you want to add an outline around a box, you can simply use `outline`.
