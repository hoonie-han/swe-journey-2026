- HTML Lesson #12: Learned about select elements, including name and value attributes.
    - Example
    
    <select name="country">
    <option value="kr">Korea</option>
    <option value="us">USA</option>
    </select>
    
    - Add multiple attribute to <select> to allow multiple selections.
    - Add selected attribute to an <option> to make it pre-selected by default.
    
    - <select> - creates a dropdown menu
    - <option> - represents each choice in the dropdown
    - value attribute - used for processing the selected data
    (often with JavaScript)
    - <textarea> - for multi-line text input
