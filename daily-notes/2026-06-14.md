## What I Learned Today

### Position — relative / absolute / sticky / fixed

- **relative** keeps the element in its original spot but moves it relative to that position by the amount you set. Use `top`, `left`, `right`, `bottom` to move it from its original position.
- **absolute** is different from relative. It is taken out of the normal document flow. By default it positions itself based on the top-level parent (`body`).
  - The position reference is the parent. If you want a specific box to be the reference parent, set `position: relative` on that box.
- **sticky** stays in its original position, then sticks (locks) at the position you set once you scroll past it.
- **fixed** is locked in a set position from the start and stays there even as you scroll.
