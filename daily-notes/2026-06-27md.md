## What I Learned Today

### Alignment Techniques

- `margin: auto` centers an element horizontally.
- `text-align: center` centers the inline elements inside a parent.
- `transform: translate(x%, y%)` moves the element itself (relative to its own size).
- `flex` makes centering much easier.

### How flex centers (the easy way)

```css
display: flex;
justify-content: center; /* centers along the main axis (horizontal by default) */
align-items: center; /* centers along the cross axis (vertical by default) */
```

- Set `display: flex` on the **parent**.
- `justify-content: center` → horizontal centering (main axis)
- `align-items: center` → vertical centering (cross axis)
- Using both → the child is centered both horizontally and vertically.
