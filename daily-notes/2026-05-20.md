## What I Learned Today

Meta Tag

**What is a meta tag?**

A <meta> tag defines **metadata** about an HTML document. Metadata means "data about data" — in other words, *information* describing the document itself rather than the content shown on the page.

**Where does it go?**

Meta tags always go **inside the <head> tag**. They are commonly used to specify:

- the character set (charset)
- the page description
- keywords
- the author of the document
- the viewport settings

**Why use meta tags?**

Because they provide information about the web page, search engines can reference them when crawling/indexing the page, and that information can be reflected in search results

**What types of attributes does a meta tag provide?**

- `charset` — It's the process of specifying a character set so text (various languages) can be displayed properly. To support both English and various languages, it's best to use **UTF-8**. <meta charset="utf-8">
- `http-equiv`— provides an HTTP header for the content's attribute information (acts like an HTTP response header)
- `name` — document information (e.g., description, keywords, author, viewport)
- `content` — the actual metadata content/value (pairs with `name` or `http-equiv`)
- `media` — a media query condition specifying which device/screen the metadata applies to (mainly used with `theme-color`)
