## What I Learned Today

CSS Selectors

Selectors = how you "target" which HTML elements to style.

1. Basic Selectors
Type — by tag name → p { }
Class — by class name → .card { } (. means class)
ID — by id name → #header { } (# means ID)
2. Group Selector
Target multiple elements at once, separated by commas.
h1, h2, p { color: black; }
3. Attribute Selector
Target by HTML attribute.
input[type="text"] { }
4. Combinator Selectors (by relationship)
Descendant — any p inside div → div p { }
Child — direct child p only → div > p { }
5. Pseudo-class (by state) — single colon :
button:hover { } → when mouse is over
li:first-child { } → first child element
Others: :focus, :last-child
6. Pseudo-element (part of element) — double colon ::
p::first-letter { } → style first letter
.box::before { content: "★"; } → insert virtual content
Others: ::after, ::first-line

## Important

Priority Order (strongest → weakest)

When multiple rules target the same element, the more specific selector wins.

1. **ID** (`#name`) — strongest
2. **Class** (`.name`) — medium
3. **Type** (tag, e.g. `p`) — weakest
