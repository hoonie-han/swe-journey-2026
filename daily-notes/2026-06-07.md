# Day 13 (2026-06-07)

## What I Learned Today

### CSS Review

- **What is CSS?** — styling the website
- Using selectors well is important!

### Cascading — Style Sheet Priority

There's an order of priority among style sheets:

- User → Code (author) → Browser

### Priority Within Code (Specificity)

- The more specific the selector, the higher its priority.
- Order (high → low): `!important` → inline → `#id` → `.class` → tag
- Avoid using `!important` when possible — it forces the style, but it's rarely used in practice.
