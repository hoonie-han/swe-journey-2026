- font-size - Define text size with number + unit.
    - px — fixed size (absolute)
    - rem — relative to <html> (most used), default 1rem = 16px
    - em — relative to parent (multiplies when nested)
    - Trick: html { font-size: 10px; } makes math easy (1rem = 10px)
    - Key: rem = based on <html>, em = based on parent

**Color**
◦ Three ways to write the same color (all = red):
◦ color: red; — color name (easy, but limited)
◦ color: `#FF0000`; — hex code ⭐ MOST USED in practice
◦ color: rgb(100%, 0%, 0%); — rgb values
◦ All three above = exact same red

- Hex code (`#FF0000`) — use this most
◦ Format: #RRGGBB (Red, Green, Blue)
◦ FF = max, 00 = none
◦ `#FF0000` = red / `#000000` = black / `#FFFFFF` = white
◦ Why most used: 16M colors possible + can copy directly from design tools (Photoshop, Figma)
- RGB
◦ rgb(red, green, blue) — use % or 0~255
◦ rgb(255, 0, 0) = same red
