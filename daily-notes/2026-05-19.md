## What I Learned Today

Sending Data - the <form> tag is used to create input forms

- <form> is the HTML tag used to send input data to a server
- A server is a host (computer) that provides information
- Server and Client
◦ When a client sends a request, the server responds with the info
◦ Example: A client requests to open Google → server → client
◦ Example: A login form has three input fields, and the input data is sent to the server through the <form> tag
- <form> wraps input elements and can submit the input values to the server.
    - input elements can used as many as you can

<form> attributes

- url: server program address
- action: the server URL where the input data will be sent
- method: how the client sends the input data (GET or POST)
- Example:
<form action="example.php" method="POST">
</form>
- In this example:
◦ The input data will be sent to the server program "example.php"
◦ The data will be sent using the POST method

GET vs POST

- GET: sends a request to the server to retrieve a response
◦ A request to "fetch" information from the server
- POST: sends a request to the server to perform an action
◦ Used to add, modify, or delete data on the server
◦ A request to "manipulate" information on the server
- Note: More details about GET and POST can be learned later, after studying HTML
