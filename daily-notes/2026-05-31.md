## What I Learned Today

**Display property Value**

| property values | meaning |
| --- | --- |
| inline | sits in a line, only as wide as content (e.g. span) |
| block | takes full width, stacks vertically (e.g. div, p) |
| inline-block | nline placement, but can add block properties (width/height) |
| none | not displayed at all (hidden) |
- **border**
    - Draws a border around an element's area
    - Sets thickness, style, color together → called "shorthand property" (단축속성)

        Example

- span { border: 2px solid green; }
- = make a 2px thick, solid, green border
- Order: thickness(2px) + style(solid) + color(green)
