## What I Learned Today

**viewport**

 The viewport means the area of a web page currently visible on the screen.

- Every device has a different viewport size (phone, tablet, laptop...)
- So the same web page can get scaled differently depending on the device
- This makes the page look bigger or smaller than intended on some screens

This is the problem the `viewport` meta tag fixes:

<meta name="viewport" content="width=device-width, initial-scale=1.0">
